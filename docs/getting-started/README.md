# Getting Started

This section covers everything you need to start using stripe402 — from installing the packages to running a working server with paid endpoints.

## Prerequisites

- **Node.js** 22 or later
- **pnpm** 10.28+ (or npm/yarn — pnpm is recommended for the monorepo)
- **Stripe account** — [sign up for free](https://dashboard.stripe.com/register) and use test mode keys
- **Redis** or **PostgreSQL** — for storing client balances (Redis is the default)
