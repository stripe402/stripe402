# Getting Started

Install the packages and get a working server with paid endpoints.

## Prerequisites

- **Stripe account** — [sign up for free](https://dashboard.stripe.com/register) and use test mode keys
- **Redis** or **PostgreSQL** — for storing client balances (Redis is the default)

> For language- and runtime-specific requirements, see the [packages documentation](../packages/README.md) for the ecosystem you're using.
